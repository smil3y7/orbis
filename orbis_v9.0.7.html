ORBIS v9 — Help

Orbis maps recurring locations across your dreams. Each bubble is a location; the app links dreams to it automatically by matching text.

Quick start

1. Load a .sqlite database, or start with an empty one.

2. Use Discovery to search your dream text for recurring places.

3. Create a Location for one, then open it and switch to Edit.

4. Set its search terms so matching dreams link automatically.

5. Pin any dreams the automatic search misses, by hand.

Map controls

Click a bubble → open its details

Drag a bubble → move it

Drag empty space → pan the view

Scroll → zoom in / out

Ctrl+click two bubbles → connect them with a Warp

Location details

👁 View — dreams, stats, warps

✏ Edit — name, type, group, search terms

Search terms → precise dream matching

📌 Pin dreams → manual linking

Bubble indicators

● Search terms are set

● Search terms are missing

Database

Any .sqlite file works, as long as it has SleepCycle/Dreams/Location tables. Orbis adds its own tables automatically:

Atlas_Nodes · Atlas_Warps
Atlas_NodeDreams
